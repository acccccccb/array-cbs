# cbs

传入二维数组，返回二维数组的所有排列组合

## 安装

```bash
npm i cbs
```

## 测试

```bash
npm run test
```

# 使用

```javascript
import cbs from 'cbs';

const arr = [
    ['黑色', '白色', '金色'],
    ['16G', '32G', '256G', '512G'],
    ['4.7', '5.5', '7.3'],
];
(async () => {
    const res = await cbs(arr);
    console.log('Input:', arr);
    console.log('Output:', res);
})()
````
